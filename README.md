## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Suhani-01/Razorpay-clone.git
   cd Razorpay-clone
   ```
2. Install dependencies :
    ```bash
    npm install
    ```
3. Run 
    ```bash
    npm start
    ```
